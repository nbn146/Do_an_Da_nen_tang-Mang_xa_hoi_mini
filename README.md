# -n-a-n-n-t-ng---M-ng-X-h-i-Mini
Đồ án tập trung nghiên cứu và phát triển một hệ thống Mạng xã hội Mini cho phép người dùng tương tác trên nhiều thiết bị khác nhau (Web và Mobile) với một trải nghiệm đồng nhất.
